# Chapter 1: 不可思议的思维切换

## Core Idea
Every language is a code that changes what information the brain activates. Switching language can change attention, memory, emotion, decision-making, perception, and identity.

## Frameworks Introduced
- **Language as code**
  - When to use: Explaining why a new language changes cognition rather than merely adding vocabulary.
  - How: Identify what the code segments, names, values, and makes easier to retrieve.
- **Language-dependent memory**
  - When to use: Legal interviews, therapy, autobiographical memory, or multilingual education.
  - How: Match retrieval language to the language of the original event or learning context.
- **Foreign language effect**
  - When to use: Moral, financial, medical, or risk decisions made in a non-native language.
  - How: Check whether emotional distance is changing the decision frame.

## Key Concepts
- **Symbol system**: A set of labels and rules for compressing and communicating meaning.
- **Metalinguistic awareness**: Awareness that objects and labels are separable.
- **Perception filter**: The way available language categories change what is noticed or remembered.

## Mental Models
- Use the Boggle-board model: rotating the same material creates new possible readings.
- Think of multilingualism as having several maps for one territory; each map reveals different routes.

## Anti-patterns
- **Vocabulary-only view**: Treating language learning as memorizing equivalent word lists.
- **Single-self assumption**: Assuming one language shift cannot change emotional or moral stance.

## Worked Example
In bilingual memory tasks, people retrieve different autobiographical events depending on the language used in the prompt. The same question asked in Cantonese or English can surface different playground memories because each language is linked to different life contexts.

## Key Takeaways
1. Language can change which memories and identities become accessible first.
2. Native language often carries stronger emotional force than a later-learned language.
3. A second language can create useful distance, but distance is not the same as objectivity.
4. Labels shape both perception and action.
