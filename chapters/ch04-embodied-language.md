# Chapter 4: 语言造就的肉身

## Core Idea
Language experience becomes embodied: it changes distributed brain networks, structural connectivity, neural efficiency, metabolic patterns, and possibly gene expression pathways.

## Frameworks Introduced
- **Distributed language network**
  - When to use: Rejecting the idea that each language lives in a separate brain location.
  - How: Analyze language as whole-network orchestration across perception, meaning, control, and motor systems.
- **Emergence model**
  - When to use: Explaining how language ability arises from many interacting parts.
  - How: Treat the system as self-organizing; the whole cannot be understood by isolating one module.
- **Experience-dependent plasticity**
  - When to use: Explaining how language practice changes brain structure or efficiency.
  - How: Link repeated activation to stronger pathways, pruning, and more efficient control.

## Key Concepts
- **Grey matter**: Processing areas containing neural cell bodies.
- **White matter**: Connection pathways linking processing regions.
- **Epigenetics**: Changes in gene expression without changing DNA sequence.

## Mental Models
- Think of language learning like physical training: repeated use changes the body that performs it.
- Treat languages as different symphonies played by overlapping orchestras, not separate instruments.

## Anti-patterns
- **Brain map oversimplification**: Assigning each language to a neat location.
- **Module fantasy**: Assuming language, memory, perception, and control are sealed systems.

## Worked Example
Multilingual aphasia cases show selective loss or recovery of languages. The chapter argues this does not prove separate locations; it reflects complex, overlapping networks with different vulnerabilities.

## Key Takeaways
1. Multilingualism changes both language and non-language networks.
2. Brain efficiency can increase as language-control demands become practiced.
3. Language is intangible, but its effects are materially embodied.
4. The strongest model is networked, dynamic, and self-organizing.
