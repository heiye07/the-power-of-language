# Chapter 3: 创意、知觉和思维

## Core Idea
Multilingual experience can increase creativity and reshape perception by creating richer links across concepts and by loosening the boundaries imposed by any one language.

## Frameworks Introduced
- **Association expansion**
  - When to use: Explaining creativity, divergent thinking, metaphor, or problem solving.
  - How: Look for cross-language overlaps that connect concepts a monolingual network would keep apart.
- **Weak linguistic relativity**
  - When to use: Discussing how language influences color, time, space, or sensory experience.
  - How: Avoid determinism; ask how language changes speed, memory, salience, and default interpretation.
- **Perception as neural prediction**
  - When to use: Explaining illusions or sensory differences.
  - How: Ask which prior experience makes one interpretation more likely than another.

## Key Concepts
- **Sapir-Whorf hypothesis**: Language shapes thought; strong determinism is too rigid, but weaker relativity has evidence.
- **Multisensory integration**: Combining visual, auditory, and other inputs into one perception.
- **McGurk effect**: Visual speech changes heard speech.

## Mental Models
- Use language as an extra aperture: more languages give more routes into the same world.
- Treat perception as a negotiation between input and prior activation.

## Anti-patterns
- **Strong determinism**: Claiming people cannot think what their language does not name.
- **Universal perception assumption**: Assuming identical stimuli always create identical experience.

## Worked Example
In color and time experiments, speakers of languages with different labels or metaphors respond differently in speed or memory. The effect is not inability; it is altered salience and encoding.

## Key Takeaways
1. Multilinguals can form more remote associations because their concepts overlap in more ways.
2. Language influences perception without fully determining it.
3. Creative advantage often comes from seeing relationships others do not notice.
4. Sensory experience is also shaped by language history.
