# Cheatsheet

## Decision Rules

| Situation | Use This Rule |
|---|---|
| A multilingual person recalls an event poorly | Try the language active during the original event before assuming memory failure. |
| A decision was made in a second language | Check whether emotional distance or framing resistance changed the decision. |
| A child hears two languages at home | Do not treat multilingual input as confusion. Assess input quality, quantity, and interaction. |
| A clinical or school assessment flags a language issue | Test whether the issue appears across all relevant languages before calling it a disorder. |
| A phrase feels politically loaded | Rename the frame and compare reactions: what does each label make salient? |
| A translation sounds correct but fails socially | Rebuild cultural function, not just dictionary meaning. |
| A research result claims to explain human cognition | Check whether multilingual and non-WEIRD participants were included or controlled for. |
| A learner is choosing when to start a language | Best time is birth; second best time is now. |

## Red Flags

- Treating monolingual cognition as the default human mind.
- Treating language difference, accent, or dialect as impairment.
- Translating literally when the task depends on humor, risk, emotion, or culture.
- Assuming a second-language answer carries the same emotional weight as a native-language answer.
- Ignoring language history in memory, legal, therapy, or education contexts.
- Assuming language does not matter because a concept is technically translatable.
